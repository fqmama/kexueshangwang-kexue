<h2 class="wp-block-heading" id="purevpn速度测试"><span id="PureVPN-2">PureVPN速度测试</span></h2>
<p>油管播放速度测试，是我对每个VPN做测评的时候，必须做的一个环节，因为我们<a href="/gfw/">翻墙和科学上网</a>大多时候都需要观看视频。我个人对一个VPN的最低要求是，至少能够播放1080P的视频清晰度，这是一个付费VPN必须要做到的。</p>
<p>显然PureVPN的Youtube播放测试结果，没有让我失望，播放4K视频不需要缓冲，非常流畅。即便我随意切换画质，或者拖动进度条，依然能够保持这种流畅程度。</p>

<p>正如我之前提到的，当你连接到PureVPN 服务器时，你的连接是加密的。如果有人正在始监视你的网络，他们将看到的只是被加密的内容。PureVPN 使用<strong>AES-256-CBC</strong>密码来确保是这种安全性。许多安全 VPN使用相同的方法来保护连接，因此该服务不是特例。</p>

<h3 class="wp-block-heading" id="vpn协议">VPN协议</h3>
<p>通常，加密不是你在使用 VPN 服务时可以修改的东西。但是，你可以选择VPN协议。这些可能会带来某些优势，在某些情况下可能非常有用。</p>
<p><strong>WireGuard</strong> – 最新和发展最快的VPN协议之一。它使用的代码行数显着减少，并提供了极快的速度。</p>
<p><strong>IKEv2</strong> – 一种较旧的协议，由于其广泛的兼容性而仍然被广泛使用。它还可以在不中断连接的情况下轻松地在 wifi 和移动网络之间切换，非常适合手机、平板电脑和其他便携式设备。</p>
<p><strong>OpenVPN</strong> <strong>(TCP/UDP)</strong> – 一种开源 VPN VPN协议，至今仍是最安全、最稳定的协议之一，尤其是在 TCP 模式下。虽然 UDP 确实牺牲了一些完整性检查周期以使其更快一些，但它因此更容易出错。</p>
<p><strong>SoftEther</strong> – 从技术上讲，这本身并不是一个VPN协议。最好将其描述为使用 SSL over HTTPS 进行数据传输的任何防火墙中打孔的方法。它在限制性制度或严格的住宅网关防火墙后面最有用。不过，它是加密方面最不强大的选项。</p>
<p><strong>SSTP</strong> – 由 Microsoft 制造，此VPN协议是 Windows 独有的。它曾经因其绕过防火墙的能力而受到称赞，也是翻墙科学上网不错的VPN软件协议。</p>
<p><strong>Bolt</strong> – PureVPN上最新的VPN协议。非常安全，并且提供了令人难以置信的速度，可以在互联网审查的国家使用。</p>

<p>第一次测试产生了好坏参半的结果。在路由器上阻止 IP 地址后，ping 命令仍在运行，这意味着并非所有内容都被阻止。工作中的终止开关会显示无法访问或超时错误，而这里的情况并非如此。DNS 功能被完全禁用，尽管如此，它并没有涵盖所有可能在你的连接断开时可能被利用的漏洞。</p>
<p>第二次测试也进行了类似的测试，在终止进程和启动终止开关之间也有一个短暂的窗口。这仍然留下了一个小的风险窗口，这不是最佳的。</p>
<p>两项测试都表明PureVPN的终止<strong>开关不是防故障</strong>的，因此如果你将 VPN 用于高敏感度任务，则应注意。可能存在终止开关不足的情况。</p>
<h3 class="wp-block-heading" id="日志记录">日志记录</h3>
<p>PureVPN 位于英属维京群岛。它不仅对于 VPN 服务提供商来说是一个有点异国情调的地方，而且对于<strong>五眼联盟</strong>来说也是遥不可及的。英属维京群岛没有数据保留法，因此 VPN 服务提供商不需要存储它们。这允许真正的匿名服务。</p>
<p>根据PureVPN的隐私政策，该服务就是这样 – 完全匿名，并且不保留任何日志。该政策规定他们不保留任何类型的连接信息或交换内容。因此，无论你打算在网上做什么，都不会留下痕迹。</p>
<h2 class="wp-block-heading" id="服务器和位置"><span id="i">服务器和位置</span></h2>
<p>PureVPN 拥有大约<strong>6000+台服务器</strong>，分布在多个个国家/地区，以下是地点的细分：</p>
<figure class="wp-block-table"><table><thead><tr><th>地区</th><th>国家数量</th></tr></thead><tbody><tr><td>欧洲</td><td>27个国家</td></tr><tr><td>美洲</td><td>4个国家</td></tr><tr><td>亚太地区/其他</td><td>16个国家</td></tr></tbody></table></figure>
<p>虽然在美洲只有四个国家有服务器，但它们在这些国家拥有一些服务器位置最多的地方。换句话说，如果你想连接到美国，这不会有任何问题，但如果你需要在南美一个较小的国家，你可能会遇到问题。</p>
<h2 class="wp-block-heading" id="界面和易用性"><span id="i-2">界面和易用性</span></h2>
<h3 class="wp-block-heading" id="windows">Windows</h3>
<p>乍一看，该电脑VPN应用程序看起来相当小，但是一旦你进入设置，它就会打开许多方法来自定义你的 VPN 连接。</p>
<p>有一些罕见但有用的功能，例如<strong>启用自动协议切换</strong>的能力。如果你遇到连接问题，你甚至可以选择允许此功能使用的协议。</p>
<p>它有 Stealth Guard，这是一种特定于应用程序的 kill switch，但它可以同时与系统范围的 kill switch 一起运行。它还允许你<strong>在终止开关启动时启动自定义脚本</strong>。这是高级用户肯定会喜欢的。</p>
<p>我发现在服务器选择中，你无需单击搜索框即可开始输入要连接的位置的名称，这很方便。</p>
<h3 class="wp-block-heading" id="mac">Mac</h3>
<p>该应用程序的外观和功能类似于 Windows 应用程序。Mac VPN甚至像 Windows 应用程序一样支持拆分隧道，尽管在使用终止开关时<strong>无法启动脚本。</strong></p>
<p>我很高兴看到包含 WireGuard，因为有时它仍然不适用于 macOS 版本。不过，该版本的应用程序中没有 SoftEther，这意味着来自 VPN 受限国家的一些用户可能会感到失望。</p>
<h3 class="wp-block-heading" id="linux">Linux</h3>
<p>有关 Linux 应用程序的信息到处都是。在他们的页面上，他们声明他们有一个适用于 Linux 的应用程序，但是当你单击该链接时，它会将你带到一个解释如何设置手动连接的页面。我发现一篇博文说你可以从他们的 GitHub 页面下载该应用程序，但该页面上的文档很少。</p>
<h3 class="wp-block-heading" id="手机vpn">手机VPN</h3>
<p>PureVPN 的手机vpn版本具有简约的 UI。有人可能会说这有点太简单了，因为即使大多数提示都借用了默认的 Android 元素。好处是它不会消耗太多资源，尽管我见过更漂亮的解决方案。</p>
<p>与桌面客户端相比，设置中的选项更少，如果你关闭应用程序，它会断开连接。通知栏中没有指示你已连接的图标。这有点烦人，因为每次你都必须打开应用程序并将其最小化以检查你的连接状态。</p>
<p>与 Android 应用程序借用默认元素的方式类似，PureVPN iOS 应用程序使用通用 iOS 应用程序模板。看起来没什么特别的，但是如果你需要快速找到一些东西，它是相当直观的。另外，你可以在明暗模式之间切换。</p>
<p>该应用程序甚至添加了 WireGuard VPN协议，因此即使是移动版本也没有被查看。此外，可以使用终止开关，这并不常见。但是，我没有测试它的 iOS 实现。</p>
<p>否则，PureVPN 的 iOS 应用似乎与 Android 版本存在类似的缺陷。这两个应用程序都非常小，不显示连接信息。</p>
<h3 class="wp-block-heading" id="浏览器扩展">浏览器扩展</h3>
<p>PureVPN 还为 Chrome 和 Firefox 提供免费扩展。他们只有 3 个位置可供选择，如果启用 Socks 连接，则根本无法选择位置。它还会禁用 WebRTC，并且可以在你每次打开浏览器时自动连接。</p>
<h2 class="wp-block-heading" id="客户支持"><span id="i-3">客户支持</span></h2>
<p>该提供商的客户支持从<strong>知识库</strong>开始。在那里，你会找到各种场景的指南，从最基本到高级。</p>
<p>如果你想与人取得联系，PureVPN 不会让你失望。<strong>我通过实时聊天</strong>联系了客户支持以解决 WireGuard 问题。在问了一些一般性问题后，他们指示我<strong>创建一个工单</strong>，创建工单大约 3 分钟后得到答复。在线客服更是实时回应，我真的很惊讶他们的反应速度。</p>
<h2 class="wp-block-heading" id="我是否推荐purevpn"><span id="PureVPN-11">我是否推荐PureVPN？</span></h2>
<p>是的。我认为PureVPN在中国翻墙还不错，也是我非常喜欢的VPN。</p>
<p>PureVPN提供了大多数人都负担得起的价格，并且具有许多的服务器选择。他们的客户服务部门给了我很好印象，并且在Netflix上的成功率非常高。这些都是PureVPN值得表扬的地方。</p>
<p>当然PureVPN也有一些问题，比如带宽不大，对于需要用<a href="/vpn-download/">VPN下载</a>大文件的用户可能不是很友好。但是观看youtube高清视频没有问题，所以也不用担心，平且它的ping值很低，这样也就平衡了带宽小的缺点。</p>
